# 🍃 NoSQL Database Systems & Schema Design
**Practical Implementations using MongoDB & Mongoose**

This repository focuses on database architecture, demonstrating how to build scalable and efficient data models for different business use cases using MongoDB.

---

### 📂 Key Implementations

#### 🛒 Product-CRUD System
- **Objective:** Managing inventory data with full Create, Read, Update, and Delete functionality.
- **Concepts:** Schema validation, data types, and filtering logic.

#### 🎓 Student Management System
- **Objective:** A relational-style data model within a NoSQL environment.
- **Concepts:** Handling student records, course enrollments, and status tracking.

---

### 🛠️ Core Skills Demonstrated
* **Schema Modeling:** Designing flexible yet robust document structures.
* **Query Optimization:** Efficiently retrieving data using MongoDB query operators.
* **Backend Integration:** Connecting database logic with Node.js/Express.
* **Error Handling:** Implementing validation layers to ensure data integrity.

---

### 🚀 Getting Started
To test these schemas locally:
1. Ensure you have **MongoDB** installed or a **MongoDB Atlas** URI.
2. Clone the repo: `git clone https://github.com/AshitaSharma01/MongoDB.git`
3. Run `npm install` and check the connection strings in the config.
